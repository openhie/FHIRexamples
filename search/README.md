# FHIR search

This repository contains several FHIR examples for experimenting with FHIR Search

The FHIR Search specification is available at [https://hl7.org/fhir/search.html](https://hl7.org/fhir/search.html)  

GraphQL is available under [https://hl7.org/fhir/graphql.html](https://hl7.org/fhir/graphql.html)

You can read these examples or you can execute them in VSCode by installing the extension [humao.rest-client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client)
